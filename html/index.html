基于RaspberryPi Zero + Zero Key + 扩展板ST7789 实现多功能U盘

功能简介

单功能实现

  • 迷你编译环境
  • Python自带web服务器
  • 扩展板菜单栏显示与选择
  • U盘模式
  • 网卡模式
  • AP模式
  • BadUSB 模式

组合功能

  • 扩展板菜单栏显示与选择 + BadUSB模式：实现脚本化键盘输入
  • 扩展板菜单栏显示与选择 + AP模式 + BadUSB模式：实现手机端控制电脑键盘输入
  • 扩展板菜单栏显示与选择 + U盘模式 + 迷你编译环境和文件导出
  • …

菜单详情

  1. Get_Status 获取当前树莓派状态，如IP，温度，服务状态等
  2. ChangeMode 切换模式，如U盘模式，网卡模式等
  3. Flash_U_Disk 刷新映射出去的U盘的文件内容
  4. Re-GetIP 重新获取IP或查看IP
  5. CatCmdline 查看cmdline配置信息
  6. CatWlanCfg 查看Wlan配置文件
  7. GetWlanCfg_mnt2etc 复制mnt下Wlan配置文件到etc目录
  8. GetWlanCfg_etc2boot 复制etc下Wlan配置文件到boot目录
  9. GetWlanCfg_etc2mnt 复制etc下Wlan配置文件到mnt目录
  10. PI-as-keyboard 开启BadUSB功能
  11. EnableAP 设置RaspberryPi无线网卡为AP模式
  12. ExportLog 导出log到mnt下，切换为U盘模式后可导出
  13. REBOOT 重启
  14. POWEROFF 关机
  15. EXIT 退出
